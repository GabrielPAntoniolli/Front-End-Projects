What is CSS?

CSS is short for Cascading Style Sheets. CSS is used to personalize elements written by a markup language such as HTML. CSS splits the content from the visual representation of a webpage. using CSS, its possible to change the color of text and the background, font, paragraph spacing, font-size, font-family and many other functionalities.

The difference between a webpage that implements CSS and another that doesn't is huge. you have probably already seen a website that does not complete load or or has a white background with black and blue text. This happens because probably the CSS part was not loaded or it doesnt exist. And this is how a webpage with only HTML elements looks like. And its not pretty, is it? CSS enables you to personalize everything in a different archive. this way the style is separated from the HTML structure, which also contributes for a cleaner code.
